# NextJS Skill

## Description
This skill enables you to develop applications using NextJS, a popular React framework for building production-ready web applications.

## Capabilities
- Create NextJS pages and components
- Implement routing using NextJS App Router
- Handle server-side rendering (SSR) and static site generation (SSG)
- Integrate with backend APIs
- Manage state using React hooks and context
- Apply styling with CSS modules, Tailwind CSS, or other solutions
- Implement form handling and validation
- Add authentication and authorization features
- Optimize performance and SEO

## Best Practices
- Follow NextJS conventions for file structure
- Use the App Router for new projects
- Leverage NextJS built-in features like Image optimization
- Implement proper error handling
- Use TypeScript for type safety
- Follow accessibility guidelines
- Optimize for Core Web Vitals

## Common Commands
- `npx create-next-app@latest` - Create a new NextJS app
- `npm run dev` - Start development server
- `npm run build` - Build the application
- `npm run start` - Start production server
